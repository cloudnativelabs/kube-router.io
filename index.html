Intro to Kube Router

Kube-router is a distributed load balancer, firewall and router designed for operational simplicity and high performance within Kubernetes.

Just One DaemonSet/Binary

With all features enabled, kube-router becomes a lean but powerful alternative to the typical Kubernetes cluster.

Small Footprint

Check out kube-router's tiny codebase! Kube-router leverages the IPVS that is already in a Kubernetes node. Add in GoBGP's modern BGP interface, and you got a recipe for awesome-sauce networking magic.

High Performance

Performance is the main motivation for developing kube-router. Combining BGP and IPVS is a perfect recipe for high-performance cluster networking at scale. BGP ensures that the data path is dynamic and efficient while IPVS provides in-kernal load balancing that has been thoroughly tested and optimized.
